Detailed Description

C99 extern declarations for static inline functions within helm.h.

When a header-only library uses static inline, each translation unit gets its own private copy. Providing matching extern declarations in a single .c file lets the compiler emit one shared, externally-visible definition that the linker can use when inlining is not performed.
